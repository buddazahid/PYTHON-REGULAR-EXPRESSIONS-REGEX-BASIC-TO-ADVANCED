# DAILY TASK — Python Tutorial Scripts

This repository contains Python tutorial scripts that demonstrate practical standard library usage for learning and basic project building.

## Included Script

- `PYTHON REGULAR EXPRESSIONS (REGEX) BASIC TO ADVANCED.py`
  - Covers the `re` module for pattern matching, validation, searching, replacement, and text cleaning.
  - Includes examples for email validation, phone validation, password validation, URL extraction, data cleaning, and log analysis.

## What You'll Learn

- Regex basics: search, match, findall, split, and sub
- Pattern syntax: character classes, quantifiers, anchors, shorthand classes, and groups
- Data validation examples for email, phone, password, USN, and vehicle numbers
- Text processing examples for extracting emails, numbers, URLs, and removing special characters or extra spaces
- Real-world scripting examples such as resume parsing and log file analysis

## Getting Started

### Prerequisites

- Python 3.8 or newer
- No external dependencies required

### Run the Script

```bash
python "PYTHON REGULAR EXPRESSIONS (REGEX) BASIC TO ADVANCED.py"
```

## Notes

- This repository is designed for learners who want to explore Python regular expressions.
- The script prints clearly labeled output sections to make it easy to follow each example.
- You can modify patterns or add new regex examples to extend the tutorial.
